# Diplomarbeit – (SOLL-Version)

Dieses Repository enthält die im Rahmen der Diplomarbeit entwickelte Sicherheitsstufe für die bestehende GitLab CI/CD-Pipeline.

## Inhalt
- Erweiterte GitLab CI/CD-Pipeline mit **Security Stage**  
- Integration von **TFLint** und **Trivy** zur Sicherheitsprüfung  
- Implementiertes **Pre-Commit Hook**-System mit globaler Konfiguration  
- Zentrale Skripte für automatisierte Code-Scans (`check_security_scan.sh`)  
- Projektbezogene Konfigurationen unter `security-conf/`

## Hinweis
Aus Datenschutz- und Unternehmensgründen wurde der Quellcode zensiert.

Dieses Repository dient ausschliesslich der Dokumentation im Rahmen der Diplomarbeit an der TEKO Olten.
